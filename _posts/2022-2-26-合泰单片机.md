### 延时

```c++
void DelayXuS(u32 uS)

{

 uS *= 20;

 while (uS--);

}
```

```c++
/*********************************************************************************************************//**

 \* @brief  Delay function.

 \* @retval None

 ************************************************************************************************************/

static void Delay(vu32 count)

{

 while (count--);

}
```
